OpenWISP Features

OpenWISP offers a wealth of features, modules, and reusable code to help you meet all your network management needs.

Configuration Templates

Effortlessly manage device settings by defining reusable configuration templates that apply updates system-wide with a single change.

Configuration Templates and Variables

Once defined, templates can be applied in the desired order either manually, as shown below, or automatically using default templates, tags, or group templates.

Variables can be used to define configuration segments that can be customized for each device, group, or organization.

Automatic Provisioning

Seamlessly connect and configure new devices with zero-touch auto registration, enabling rapid deployments with minimal manual effort.

Automatic VPN Tunnels

Automatically provision secure VPN tunnels with all required configurations, keys, and certificates, reducing errors and speeding up node deployment.

Network Monitoring

Monitor network performance with insights into latency, reachability, hardware usage, traffic patterns, and Wi-Fi clients for proactive management.

Alerts & Notifications

Receive timely notifications thanks to configurable alerts, tailored to your network's needs for fast issue detection and response.

Multi-tenancy

Isolate tenants within separate organizations, ensuring each has access only to their own data and configurations, while super administrators maintain full oversight.

Firmware Upgrades

Efficiently manage firmware upgrades with options for individual or batch updates, keeping your devices current with the latest features and security patches.

Network Topology

Visualize and manage your network’s layout with comprehensive topology maps, offering clear insights into device connections and network structure.

WiFi & Roaming

Optimize network performance with seamless WiFi roaming and intelligent access point management, ensuring consistent connectivity and broad coverage.

Hotspots & Public WiFi

Deploy hotspots and public WiFi with captive portal integration for registration, authentication, social login, mobile phone number verification and paid subscriptions.

Node management

Whether you have a team of 2 or 200, our shared team inboxes keep everyone on the same page and in the loop.

Mesh networks

Enhance network reliability with mesh networks, providing robust, self-healing connections that ensure seamless coverage and optimal performance.

RADIUS & EAP

Adopt RADIUS to handle authentication, authorization, and accounting, ensuring secure and efficient access control across your network.

IPAM

The IPAM module automatically provisions subnets and IP addresses based on allocation rules set by network administrators.
