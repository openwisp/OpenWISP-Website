Contributing

Please keep in mind that if you wish to contribute to OpenWISP, you should use GitHub for doing so, and:
  • Fork the application you wish to contribute to.
  • Add new code respecting as much as possible the code style and formatting conventions used. If you wish to modify existing code, then please try to isolate your modifications as much as possible.
  • Send us a pull request.

Patches that are submitted individually (i.e., outside GitHub) are likely to be ignored.
