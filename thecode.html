Source Code

OpenWISP is a modular platform which is composed of many modules and libraries. For this reason there is no single OpenWISP code repository but rather a repository for each module.

The source code of these modules can be found on github.com/openwisp.

Each module has its own documentation but it's usually aimed at developers.

There's also a comprehensive user documentation website which the community is contributing to improve over time.
